# mugenKuma

> Unity で自作した熊を無限に生成し続けるプロジェクト

## 概要

Unity 上で自作した熊の 3D モデルを、ボタン操作で無限に `Instantiate` し続けるプロジェクトです。

- A キーを押すたびに熊オブジェクトが生成される
- 熊の 3D モデル（`kumasan0320.obj`）は自作

## 使用技術

- Unity（C#）
- Unity 3D（Instantiate / GameObject）
- 自作 3D モデル（OBJ 形式）

## 動作環境 / 注意事項

- Unity でプロジェクトを開いて実行してください
- 動作確認済みの Unity バージョンは不明です
